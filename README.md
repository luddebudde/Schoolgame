# Schoolgame

Schoolgame now has a Vite + React + TypeScript starter with a simple TSX-based interface.

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

   Open the local URL that Vite prints in the terminal.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates the production files in `dist/`.
- `npm run preview` serves the built `dist/` output locally after `npm run build`.
- `npm run lint` runs ESLint.
